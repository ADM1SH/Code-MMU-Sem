# Multimedia University (MMU) Semester Projects

## Description
A central hub for various programming projects and laboratory work completed during my semesters at MMU. This repository showcases a range of skills from low-level assembly to high-level object-oriented programming.

## Highlighted Projects
- **Booking System**: A C++ project managing facility bookings, demonstrating CRUD operations and file handling.
- **Assembly Labs**: ARM/x86 assembly exercises focused on registers, memory management, and basic arithmetic.
- **C-CPP Coursework**: A collection of programs ranging from basic logic to advanced data structures.

## Folders
- `Assembly-Labs/`: Low-level architectural exercises.
- `Booking-System-Project/`: Source code for the core semester project.
- `C-CPP-Coursework/`: Practical lab work and exam preparation scripts.
- `Virus.DOS/`: Educational samples of legacy DOS-based code.
